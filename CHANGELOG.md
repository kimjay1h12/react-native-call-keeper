# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-10-14

### Added
- Initial release of react-native-call-keeper
- CallKit integration for iOS
- ConnectionService integration for Android
- New Architecture (TurboModules) support
- Expo config plugin for easy integration
- Full TypeScript support
- Display incoming call notifications
- Start outgoing calls
- Answer/Reject call functionality
- Mute/Unmute support
- Hold/Resume support
- DTMF tone support
- Video call support
- Event-driven architecture
- Comprehensive documentation
- Example application

### Features
- ✅ iOS CallKit integration
- ✅ Android ConnectionService integration
- ✅ React Native New Architecture support
- ✅ Expo compatibility
- ✅ TypeScript definitions
- ✅ Full call management (incoming, outgoing, answer, reject, end)
- ✅ Call controls (mute, hold, DTMF)
- ✅ Event listeners for all call actions
- ✅ Video call support
- ✅ Permission handling
- ✅ Comprehensive example app

### Supported Platforms
- iOS 13.0+
- Android API 23+
- React Native 0.70+
- Expo SDK 48+
